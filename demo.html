API Endpoints

Two simple endpoints, zero ceremony.

GET/cards

Returns the entire deck of 78 cards with names, descriptions, and image URLs.

GET/cards/onecard

Returns a single random card. Great for daily readings or "card of the day" widgets.

GET/tarotdeck/:image

Serves the high resolution card artwork directly. e.g. /tarotdeck/thefool.jpeg

Try It in Code

Works with anything that can speak HTTP. No SDK needed.

JavaScript // Pull a single random card const res = await fetch("http://localhost:3000/cards/onecard"); const card = await res.json(); console.log(card.name); // "The Star" console.log(card.description); // Full interpretation console.log(card.image); // "/tarotdeck/thestar.jpeg"
